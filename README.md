# Slideshow

A simple programmable slideshow application in Svelte.

## Features:

Slide Types:

* Title slide
* Text slide
* Graphic slide
* Embed slide
* Multi-slide (any pair of text, graphic, or embed)

Slide Events:

* OnEntry
  * Fade in text/graphic
* OnExit
  * Fade out text/graphic
